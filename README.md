# dotfiles

Basics:

* tmux
* zsh
* $EDITOR=(n)vim

Extras:

* z
* fzf


## Installation

```
cd ~
git clone https://github.com/eemp/dotfiles
cd dotfiles
make all
chsh -s zsh
```
